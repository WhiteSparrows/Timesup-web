# Timesup - Jeu de Devinettes

Une application simple pour jouer au jeu Timesup avec 2 équipes, 3 manches et un système de comptage de points.

## Comment jouer

1. Ouvrez `index.html` dans un navigateur web
2. Choisissez le nombre de mots (entre 5 et 50)
3. Cliquez sur "Commencer le jeu"

## Règles des 3 manches

- **Manche 1** : Décrivez le mot avec des phrases (sans dire le mot lui-même)
- **Manche 2** : Décrivez le mot avec un seul mot
- **Manche 3** : Mimez le mot (pas de parole)

## Contrôles

- **✓ Mot trouvé** : Valide un mot deviné (+1 point)
- **⊘ Passer** : Passe au mot suivant (0 point)
- **Fin de manche** : Termine la manche avant la fin du timer

Chaque manche dure 60 secondes.

## Caractéristiques

- ✅ 2 équipes avec système de points
- ✅ 3 manches avec règles différentes
- ✅ Timer interactif
- ✅ Design responsive
- ✅ 100+ mots prédéfinis
- ✅ Aucune dépendance, fonctionne en local
